# llm-lab

> LLM 原理与工程实践的个人学习记录。
>
> 策略：基础只补到够用 → 尽快进入源码 → RL 只学大模型真正常用的那一支

---

## 四个阶段

| 阶段 | 主题 | 核心目标 |
|------|------|---------|
| Phase 1 | 基础准备 | 手写 autograd，跑通最小训练闭环 |
| Phase 2 | Transformer | 读懂 nanoGPT，理解 Attention 每个模块 |
| Phase 3 | 后训练 | 独立完成 SFT / LoRA，读懂 veRL |
| Phase 4 | 应用方向 | RAG、Agent、推理优化 |

## 核心资源

- [Karpathy nn-zero-to-hero](https://github.com/karpathy/nn-zero-to-hero) — 最佳入门视频
- [Stanford CS336](https://cs336.stanford.edu/spring2025/index.html) — 从头构建语言模型
- [d2l.ai](https://zh.d2l.ai) — 动手学深度学习
- [veRL](https://github.com/volcengine/verl) — RL 训练框架
